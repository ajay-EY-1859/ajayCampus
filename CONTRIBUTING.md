## 🚀 How to Contribute

1. **Fork the repo**
2. **Clone your fork**
3. Create a new branch (`git checkout -b feature-name`)
4. Make and test your changes
5. Commit with descriptive messages
6. Push to your fork (`git push origin feature-name`)
7. Create a Pull Request from GitHub

---

## 🧪 Suggesting Features

Have an idea for:
- 🧾 Additional dashboard items?
- 🛡️ Admin login and role-based access?
- 📤 PDF formatting improvements?

Create an [Issue](https://github.com/ajay-EY-1859/ajcampus/issues) and tag it as a `feature request`.

---

## ✅ Coding Guidelines

- Use C11 standard
- Keep functions modular and reusable
- Comment major steps and edge-case handling
- Follow naming consistency (`camelCase` or `snake_case` as module demands)
